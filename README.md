## Installation

CocoaLogSocket is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
use_frameworks!
target :test do
    platform :ios, '8.0'
    inhibit_all_warnings!
    pod 'CocoaLogSocket'
end

```
